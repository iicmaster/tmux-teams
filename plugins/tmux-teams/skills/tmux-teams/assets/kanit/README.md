# Embedded Kanit font

Pulse embeds the official Google Fonts Kanit v17 WOFF2 subsets so generated
HTML remains self-contained and works offline. It carries weights 400, 500, and
600 for both Thai and basic Latin.

Official CSS source:
https://fonts.googleapis.com/css2?family=Kanit:wght@400;500;600&display=swap
